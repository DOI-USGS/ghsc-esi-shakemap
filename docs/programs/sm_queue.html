sm_queue¶

Run a daemon process to accept origin, rupture, fault, etc., messages and run shake on the event.

usage: sm_queue [-h] [-a] [-b]

Named Arguments¶

-a, --attached

Inhibit daemonization and remain attached to the terminal.

-b, --break_lock

If the pid lock file can’t be acquired, break the existing lock and acquire a new lock. Do this only if you are certain tha no other sm_queue processes are running.
